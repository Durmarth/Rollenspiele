# Wanderregeln#
Laut Regeln:
1 Runde = 1 Stunde = 1 km = 1 Erschöpfungspunkt.
Alle 3 Erschöpfungspunkte zu erschöpft um weiter zu gehen
4 Stunden Pause: Erschöpfung=0

Regeln:
1. Unterstützen: 1 Bonuswürfel/verliert keinen Erschöpfungspunkt
2. Mühelos: verliert keine Erschöpfung
3. Ablenkung: Man darf forcieren
4. Abkürzung: 2 km in kürzerer Zeit zurücklegen/sonst verlaufen

Versuchte Regeln von mir:
Vier Würfe:
1. Orientierung -> Orientierung
2. Kreaturen um euch herum -> Naturkunde?
3. Passt auf Gruppe auf?

Überelge: Pro Zeit wie viel Randomereignisse?

Orientierung:
1. Fehlschlag 3 Randomereignisse, doppelte Zeit
2. Erfolg: 2 Randomereignisse, normale Zeit
3. Schwieriger Erfolg 1 Randomereigniss normale Zeit
4. Extremer Erfolg 0 Randomereignisse, halbe Zeit.

Naturkunde:
1. Fehlschlag: +1 Randomereignis, bei diesem keine Chance sich vorzubereiten
2. Erfolg: alles normal wie in der Zufallstabelle
3. Schwieriger Erfolg: einmal Chance sich besser vorzubereiten
4. Extremer Erfolg: -1 Randomereignis und immer Chance sich besser vorzubereiten
Aufpassen auf Gruppe:
5. Fehlschlag:
6. Erfolg:
7. Schwieriger Erfolg:
8. Extremer Erfolg:
